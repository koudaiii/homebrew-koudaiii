# homebrew-koudaiii
Homebrew recipes of koudaiii

## How to Release

- Create branch
- Set up version
- `$ openssl dgst -sha256 ~/Download/hoge.zip`
- Check `$ script/check-sha256sum https://github.com/koudaiii/hoge/releases/download/$VERSION/hoge.zip`
